# 勤怠管理アプリ Ver 4.0

スター歯科クリニック 西宮北口駅前院 向けの勤怠打刻データ作成ツールです。

## 概要

月次の勤怠打刻データ（CSV形式）を自動生成するシングルページアプリケーション（SPA）です。  
日本の祝日・定休日・振替出勤などを考慮したスケジュール管理と、Shift-JIS形式でのCSVダウンロードに対応しています。

## 主な機能

- 対象年月・従業員の選択
- 定休日設定（日曜固定 + 任意の曜日）
- 祝日の自動判定（2025〜2026年対応）
- 祝日週の振替診療日の自動設定
- カレンダーUIでの出勤/休み切り替え
- 有給・欠勤・振替休日の登録
- 休日出勤・時間変更（遅刻・早退など）の登録
- 出勤データのプレビューと集計表示
- Shift-JIS形式のCSVファイルダウンロード
- 設定のローカルストレージ保存（自動復元）

## 使用技術

- React 18（CDN / UMD）
- Tailwind CSS（CDN）
- Babel Standalone
- Font Awesome 6
- Encoding.js（Shift-JIS変換）

## デプロイ

本プロジェクトは **Cloudflare Pages** でホスティングされています。  
静的HTMLファイル1枚構成のため、追加のビルド設定は不要です。

### Cloudflare Pages 設定

| 項目 | 値 |
|------|-----|
| ビルドコマンド | （なし） |
| 出力ディレクトリ | `/`（ルート） |
| ルートディレクトリ | `/` |

## ローカルで試す

リポジトリをクローンしてブラウザで `index.html` を開くだけで動作します。

```bash
git clone https://github.com/nishikita-dc/kintai-app.git
cd kintai-app
open index.html   # macOS の場合
```
